# Neighbourhood Cafe & Bar — Website

A business website for Neighbourhood Cafe & Bar, a warm community café nestled in La Lucia, Durban. Built as a fully standalone, single-file site — no frameworks, no build tools, just open and go.

 **Live Site:** https://agent-69aab8562e806f1c734792ea--neighborhoodcafe.netlify.app/

##  Features

- Hero section with background photo and branding
- Full interactive menu (Breakfast, Lunch, Little Neighbours)
- About section with brand values
- Contact section with opening hours and embedded Google Map
- WhatsApp & call buttons
- Fully responsive design (mobile & desktop)
- No build step — opens directly in any browser

---

## 🛠 Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Fonts:** Google Fonts (Playfair Display, DM Sans)
- **Deployment:** Netlify / GitHub Pages

---

##  Getting Started

No installation needed. Simply open the file in your browser:

```bash
# Option 1 — Just double-click
neighbourhood-cafe.html

# Option 2 — Clone and open
git clone https://github.com/YOUR_USERNAME/neighbourhood-cafe.git
cd neighbourhood-cafe
open neighbourhood-cafe.html
```

##  Project Structure

```
neighbourhood-cafe/
```

Building this project gave me hands-on experience converting a React/Vite project into a portable, dependency-free HTML file. I practised responsive layout design, CSS custom properties for theming, vanilla JavaScript for tab switching and smooth scroll, and embedding assets as base64 for truly standalone deployment.
